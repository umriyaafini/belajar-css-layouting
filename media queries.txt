media queries

"Responsive design" adalah sebuah strategi untuk membuat situs yang "menjawab" browser dan peralatan yang digunakan untuk menampilkan situs agar terlihat keren bagaimanapun juga.

Media query adalah tool paling powerful untuk membuat responsive design ini. lihat contoh penerapan media query pada kode --> media-queries.html
